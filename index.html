「是不是都能線上自動申請?」

大部分申報與證明書申請已可線上(日本 NACCS、自由販売証明書、中國 CIFER/單一窗口、台灣防檢署檢疫雲、香港 GETS)。但有兩類環節依法必須實體進行,無法純自動:

① 日本端實體檢查 + 紙本檢疫證明書正本

由家畜防疫官親自檢查,核發紙本正本隨貨。→ 可由日本供貨廠商/出口商負責(見「誰負責哪段」)。

② 進口口岸的實貨查驗(中/台)

到貨後由目的地海關抽驗,無法由供應商代替;但你的進口報關行會到場配合,你只要確保文件正確即可順利放行。

出口流程(依地區)

關鍵原則:決定要走多少流程的,是「原料來源(是否含動物成分)」與「目的地是否要求證明書」,而不是產品用途。

A. 日本出口端(所有貨物共通)

搬入保稅地域→ 輸出申告(NACCS)→ 海關審查/檢查→ 輸出許可→ 裝船/裝機

含動物成分者:加做動物検疫(動物検疫所 AQS)

  1. 事前確認進口國受理狀況與家畜衛生條件
  2. 提出輸出檢查申請書(可走 NACCS 線上)
  3. 附文件:屠宰/衛生處理證明、市售品販售證明、再出口附原輸入証明影本
  4. 現物實體檢查(防疫官執行,不可線上)
  5. 核發輸出検疫証明書正本(可先發 NACCS 合格通知供通關)

日方官方文件:自由販売証明書(MAFF,僅線上,需 gBizID)· 動物検疫証明書(AQS,紙本正本)· 産地証明書(部分情形)

B. 中國大陸進口

前提:日本寵物食品尚未取得輸華准入,以下三層註冊在「國家准入」打開前無法啟動。非食品用品不受此限。
層級機關內容申請人
① 輸華准入海關總署 GACC日本+品類列入准入名單日本官方(政府對政府)
② 境外企業註冊GACC / CIFER工廠取得在華註冊號日本主管部門推薦
③ 進口飼料登記證農業農村部主糧/添加劑/保健品登記境內代理機構

每批報關:備齊檢疫許可證+日方檢疫證書 → 中國國際貿易單一窗口線上申報(關檢一表)→ 審核繳稅 → 查驗 → 中文標籤查驗 → 放行。中文標籤強制,須符合 GB 10648,禁療效宣稱。

非食品:玩具/礦物貓砂/一般用品走一般報關;植物源貓砂需植物檢疫許可;清潔洗護可能按化妝品邏輯,建議歸類預裁定。

C. 台灣進口

  1. 含偶蹄/禽成分 → 日方工廠須先經防檢署認證(目前約 9 家日本廠);取得日方動物檢疫證明書
  2. 向防檢署線上申請輸入檢疫(檢疫雲)
  3. 事前辦出進口廠商登記;寵食在 petfood.moa.gov.tw 申報(首次輸入30日內)
  4. 報關行經關港貿單一窗口申報(進口日起15天內)
  5. 海關核定 C1 免驗 / C2 補件 / C3 實體驗貨 → 繳稅放行
利多:台灣 2025/11/21 起撤除日本產食品輻射管制,輻射報告與產地證明已不需要。犬貓寵食不適用《飼料管理法》,改走《動物保護法》申報制。

D. 香港進口

自由港:寵食/玩具/貓砂/清潔用品零關稅、免進口牌照、自由進口。唯一義務:進口後 14 天內經 GETS(Tradelink/Brio/Ge-TS)電子報關,用 Form 1。含貓/狗成分或 CITES 物種成分才需 AFCD 許可;含反芻/豬/禽/奶建議附日方獸醫證明書。

EXW(出廠交貨)

供應商只把貨放工廠,出口檢疫/報關/證明書都要你自己在日本安排。最麻煩,不建議新手。

FOB / CIF(建議)

供應商負責日本出口報關、檢疫、取得檢疫證明書正本並裝船。你接手目的地進口。最常見、責任清楚

DDP(送到付稅)

供應商或其物流負責到「門」含進口清關。最省事但成本高,且對方須懂目的地法規。

    Agent Loop 與四層工程

    Agent Loop(智能體工作循環)是每個 AI 代理內部的運轉方式——不是問一次答一次,而是一個會自我修正的循環:

    目標→ 思考→ 行動→ 觀察→ 評估→ 修正↺ 再執行

    本平台的六個代理各自跑自己的 Loop,並把每輪結果沉澱回共用知識庫——這就是「自學習」的底層機制。

    四層工程把「怎麼讓 AI 真正能用」拆成四層,而本平台已分別落地:

    層一句話定義本平台對應(已落地)
    Prompt Engineering怎麼問 AI各代理提示語、供應商邀約信產生器、可重用 Skill 啟動提示語(海產版已產出)
    Context EngineeringAI 看得到什麼資料全亞洲 49 目的地准入知識庫 + 案例/教訓庫(架構 L4 知識層)
    Harness EngineeringAI 在什麼工具環境裡執行工具箱(准入查詢器/信件產生器)+ 整合連接器 + 瀏覽器/GitHub/Render + 每日排程
    Loop Engineering誰讓 AI 持續啟動·執行·驗證·修正·沉澱每日法規巡檢排程 + 進度知識代理(今天到此結束→沉澱)+ 學習回饋迴圈

    目標:能賣的獨立平台。

    任務:多語(中英日+在地語)多幣別;商品由供應商主檔上架;AI 定價/成本執行引擎;准入過濾(只對「可進口」市場顯示該品項);金流/會員/客服。

    交付:平台 MVP + 試營運。依賴:P1 商品資料。

    P3 合規與物流自動化(M4–M6)

    目標:下單到出貨全自動。

    任務:文件申辦代理依「品項×目的地」自動備證/填單;通關物流代理介接報關行/3PL/冷鏈、試算稅費、追蹤;中文/在地標籤合規;首批試出貨(台港星)。

    交付:首批成功通關出貨、SOP 沉澱。依賴:P1 工廠認證、P2 訂單流。

    P4 首波上線 + 行銷(M5–M8)

    目標:真正開賣並導流。

    任務:台港星+日內銷正式上線;行銷代理做各國在地化文案/素材、SEO/ASO、KOL、廣告投放(Meta/Google/LINE/字節);同步上架當地主流 marketplace;成效儀表板;合規行銷(避療效宣稱)。

    交付:首波營收、行銷成效報表。依賴:P2 平台、P3 出貨。

    P5 擴張·多品類·自學習(M8–M18)

    目標:規模化。

    任務:第二波(馬/韓/泰)、第三波(中東/印度)+中國跨境;加海產/食品等品類(沿用知識庫);法規情報+展覽巡檢持續回灌;學習迴圈優化;平台化成自助 SaaS。

    交付:全亞洲覆蓋、多品類、可複製平台。依賴:前述全部。
